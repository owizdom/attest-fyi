How it works

1

Fingerprint

The same probes run at temperature 0 against each provider. A swapped, distilled, or quantised engine diverges from the model it claims; the same model repeats itself, so any drift is real signal.

2

Audit

The attestation quote and image digest are verified, then weighed against what actually came out. A valid seal wrapped around the wrong engine is the headline fault.

3

Reproduce

Every probe set and transcript is content-hashed. Re-run the harness yourself and you land on the same verdict. Blind spots are published, not hidden.
